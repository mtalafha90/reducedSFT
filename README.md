# reducedSFT

**Controlled tests of a reduced response theory for nonlinear axial-dipole
modulation in Babcock–Leighton surface flux transport (SFT) models.**

The idea under test is that the final axial dipole produced by a solar cycle
of normalised amplitude `T` can be written in the reduced form

```
P(T) = a T Q(T)
```

where `a` is the linear response coefficient of the transport model and
`Q(T)` is an amplitude-dependent dipole-production efficiency built from
three quenching mechanisms:

| Factor | Form | Mechanism |
|--------|------|-----------|
| `Q_TQ(T) = 1 / (1 + b_TQ T²)` | algebraic | tilt quenching — stronger cycles produce smaller polarity separations |
| `Q_LQ(T) = exp(−[2 λ̄₀ δ + (1 − ε) δ²] / (2 λ_R²))`, `δ = b_LQ T²` | Gaussian | latitude quenching — stronger cycles emerge at higher latitudes, where flux contributes less to the dipole |
| `Q_I(T) = 1 / (1 + b_I T²)` | algebraic | inflow quenching — converging flows towards the activity belts reduce dipole build-up |

Here `λ̄₀` is the activity-weighted mean emergence latitude (the belt latitude
at cycle maximum) and `ε` a small variance correction; both come from the
closed-form average of the instantaneous quenching factor over the drifting
activity belt — see the theory document below.

A one-dimensional SFT model (advection, diffusion, optional decay, idealised
bipolar-ring sources) is integrated over a full cycle for a scan of
amplitudes `T ∈ [0.2, 2.5]` and five cases — linear, tilt, latitude, inflow,
and combined — and the measured normalised response is compared with the
reduced prediction `T Q(T) / Q(1)`. A reduced cycle map
`T_{n+1} = D₀ T_n Q(T_n)` then connects the single-cycle response to
multi-cycle behaviour, with an analytic stability classification in the
(`b_TQ`, `b_LQ`) plane.

## Requirements and running

Python 3.10+ with NumPy (1.x or 2.x) and Matplotlib:

```bash
pip install -r requirements.txt
python3 nonlinear_response_sft_experiment.py
```

Runtime is a few minutes. All outputs are written to
`figures_nonlinear_response_v2/`.

## Tests

The exact identities underlying the theory (adjoint duality, operator
conservativity, stepper equivalence, the closed-form closures, the map
algebra, the memory parameter, and the end-to-end latitude agreement) are
pinned by a regression suite that runs in about ten seconds:

```bash
pip install pytest
pytest -q tests/
```

The suite runs automatically on every push and pull request via GitHub
Actions (`.github/workflows/ci.yml`).

## Reference parameters

| Parameter | Value | Meaning |
|-----------|-------|---------|
| `u0` | 12 m s⁻¹ | meridional-flow amplitude, `u = u0 sin 2λ` |
| `eta` | 500 km² s⁻¹ | turbulent surface diffusivity |
| `tau` | ∞ | no explicit radial decay |
| `b_TQ` | 0.15 | tilt-quenching strength |
| `b_LQ` | 2.4° per unit T² | latitude-quenching strength |
| `b_I` | 0.2 | inflow-quenching strength, calibrated to the 10–25 % dipole reduction by inflows found by Talafha, Petrovay and Opitz (2025, SoPh 300, 57) |
| `λ_R` | 20° | dipole (dynamo) effectivity range |
| `λ̄₀` | 22.09° | belt latitude at cycle maximum (derived) |
| `ε` | 0.0144 | envelope variance correction (derived) |

These are effective demonstration values, not observational calibrations.

Key switches in `SFTParams`:

- `lq_reference` — `"belt"` (default): latitude quenching follows the
  instantaneous latitude of the drifting activity belt, and the theory uses
  the closed-form envelope average; `"static"`: the original form referenced
  to the fixed base latitude.
- `latitude_quenching_mode` — `"efficiency"` (default) or `"shift"`
  (physically displace the source belt poleward).
- `inflow_mode` — `"efficiency"` (default) or `"physical"` (explicit
  converging flows added to the meridional circulation).

## Key results

- With the reference parameters, the reduced form reproduces the SFT
  response to within 2 % (tilt), 1.3 % for `T ≤ 1.4` and 7 % at `T = 2.5`
  (latitude), and 0.016 absolute (combined). The linear and inflow cases
  agree to round-off, as linearity demands.
- The latitude and inflow factors validate the *linearity* of the transport;
  the tilt and (in belt mode) latitude cases are the non-trivial tests of
  the closure itself.
- For the reduced cycle map: a nonzero fixed point exists and is unique
  exactly when `D₀ > 1`; instability is only possible through period
  doubling, with criterion `q(T*) = −dlnQ/dlnT ≥ 2`. A single algebraic
  quenching can never destabilise the cycle, whereas Gaussian latitude
  quenching always does for `D₀ ≥ e` — the analytic content behind the
  structure of the stability map.
- The reduced theory with `Q > 0` describes saturation and over-quenched
  slopes (`dP/dT < 0`) but cannot describe a sign reversal of the dipole;
  sign-reversing regimes (strong explicit inflows) lie outside its domain.
- The adjoint (dipole-yield kernel) analysis makes the framework exact for
  source-side nonlinearities: the duality identity holds to 10⁻¹⁵, the
  transport model's intrinsic dynamo effectivity range is measured to be
  λ_R ≈ 29°, and replacing the envelope time-moments of the latitude
  closure by yield-weighted moments reduces its error from 6.6 × 10⁻² to
  5.5 × 10⁻⁵. The cycle map's period-doubling cascade starts exactly at the
  analytic boundary (D₀ = 2.92 for the reference quenchings).
- Physical validation: the kernel predicts the physical shift-mode and
  tilt scans exactly (10⁻¹³), the pair yield has λ_R ≈ 10.5° and changes
  sign at ≈ 25°, so the physical belt shift over-quenches to dipole
  reversal at T ≈ 1.1 — beyond any positive Q closure; an improved cubic
  tilt closure reaches 10⁻⁴; physical inflows follow the algebraic Q_I to
  T ≈ 1.2 at calibrated strengths and reverse the dipole at T = 1.4 for
  5 m s⁻¹ inflows.
- Multi-cycle SFT with dynamo feedback shows the map's memoryless
  assumption fails for τ = ∞: the remnant dipole survives with r ≈ 0.99
  per cycle (its decay proceeds only by cross-equatorial diffusion,
  exponentially slow at the reference R_m ≈ 17), so nearly half the gain
  is spent reversing the old polar field. The memory-corrected map
  T[D₀Q − r] restores the agreement: one-step prediction error drops from
  97 % to 1.3 %, and the SFT bifurcates exactly at the corrected boundary
  D₀ = 3.35 (memoryless: 2.80).
- Across the (η, u₀) plane the effectivity range obeys the quadrature law
  λ_R² = η/(2u₀R⊙) + w₀² (coefficient 1.008, scatter 2.5 %, w₀ ≈ the
  source ring width), the closure domain boundary is λ_rev ≈ 2.33 λ_R, and
  the decay time enters analytically: r(τ) = e^(−P/τ) r(∞) while λ_R and
  λ_rev are exactly τ-invariant. Observationally calibrated decay times
  (τ ≈ 5 yr) place the model in the weak-memory regime (r ≈ 0.11).

See `docs/theory_review_and_enhancement.md` for derivations, the error
budget of each closure, and the proposed manuscript revisions.
