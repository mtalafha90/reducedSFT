# reducedSFT

Controlled nonlinear-response experiments for a reduced Babcock–Leighton /
surface flux transport (SFT) theory of axial-dipole modulation.

The central reduced relation is

    P(T) = a T Q(T),

where T is the normalised cycle amplitude, P(T) the final signed axial
dipole, *a* the linear response coefficient, and Q(T) a nonlinear
dipole-production efficiency built from tilt-like, latitude-like, and
inflow-like quenching factors.

## Contents

| Path | Description |
|------|-------------|
| `nonlinear_response_sft_experiment.py` | The full experiment: 1D SFT solver on a μ = sin λ grid, controlled nonlinearity cases, reduced-theory comparison, and stability map of the reduced cycle map. |
| `figures_nonlinear_response_v2/` | Generated figures and tables (regenerated by running the script). |
| `docs/theory_review_and_enhancement.md` | Full review of the theory, the corrections applied, the enhanced (belt-consistent) latitude-quenching closure, and exact stability results for the reduced cycle map. |

## Running

Requires Python 3.10+, NumPy (1.x or 2.x) and Matplotlib:

    python3 nonlinear_response_sft_experiment.py

Runtime is a few minutes. Outputs (figures, `nonlinear_response_results_v2.csv`,
`theory_agreement_v2.csv`, `stability_map_v2.csv`) are written to
`figures_nonlinear_response_v2/`.

## Key options (`SFTParams`)

- `lq_reference`: `"belt"` (default) evaluates latitude quenching at the
  instantaneous latitude of the drifting activity belt, with the reduced
  theory using the closed-form envelope average; `"static"` reproduces the
  original static-latitude runs.
- `latitude_quenching_mode`: `"efficiency"` (default) or `"shift"`.
- `inflow_mode`: `"efficiency"` (default) or `"physical"` (explicit
  converging flows).

See `docs/theory_review_and_enhancement.md` for the derivations and the
quantitative agreement between the SFT experiments and the reduced theory.
